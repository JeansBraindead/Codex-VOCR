# Beta session logs

One file per tester per session, named `YYYY-MM-DD-<tester>.md`, using the
session log template in `docs/BETA_TESTING.md` §3a.

Do not edit past sessions after the fact — if something was wrong, note the
correction in a new session's "Free-form observations" instead, so the
analysis pass in `docs/beta/SUMMARY.md` sees an honest history.
